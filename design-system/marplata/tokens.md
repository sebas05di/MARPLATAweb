# Design Tokens

> **INHERITS FROM:** `MASTER.md`
> Complete token reference for the design system.

## Color Tokens

| Token | Value | Usage |
|-------|-------|-------|
| `--color-primary` | `#18181B` | Primary actions, text |
| `--color-secondary` | `#3F3F46` | Secondary elements |
| `--color-cta` | `#EC4899` | Call-to-action accents |
| `--color-background` | `#FAFAFA` | Page background |
| `--color-text` | `#09090B` | Body text |

## Typography Tokens

| Token | Value | Usage |
|-------|-------|-------|
| `--font-heading` | `Playfair Display` | Headlines |
| `--font-body` | `Inter` | Body copy |

## Spacing Tokens

| Token | Value | Usage |
|-------|-------|-------|
| `--space-xs` | `4px` | Tight gaps |
| `--space-sm` | `8px` | Icon gaps, inline spacing |
| `--space-md` | `16px` | Standard padding |
| `--space-lg` | `24px` | Section padding |
| `--space-xl` | `32px` | Large gaps |
| `--space-2xl` | `48px` | Section margins |
| `--space-3xl` | `64px` | Hero padding |

## Shadow Tokens

| Token | Value | Usage |
|-------|-------|-------|
| `--shadow-sm` | `0 1px 2px rgba(0,0,0,0.05)` | Subtle lift |
| `--shadow-md` | `0 4px 6px rgba(0,0,0,0.1)` | Cards, buttons |
| `--shadow-lg` | `0 10px 15px rgba(0,0,0,0.1)` | Modals, dropdowns |
| `--shadow-xl` | `0 20px 25px rgba(0,0,0,0.15)` | Hero images, featured cards |
